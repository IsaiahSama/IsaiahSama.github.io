Text Guided System (TGS)

A script that is based off of SMITE's VGS (Voice Guided System). This script will accept certain commands from the keyboard, and will then output the appropiate, corresponding message. The main use for this is in Fast Paced MOBA Games like LOL, which do not have a feature similar to SMITE'S VGS.

Showcase: Text Guided System

Repo: Text Guided System

Pyautogui Creations

Pyautogui is a library which allows for automation of your computer. Such as controlling the mouse, keyboard, even recognizing images on the screen. This repository of mine is a few of my projects that I would have created using this library, mainly to automate playing simple games.

Repo: Automation

Schedule Keeper

A Schedule Keeper I made using Python. One of my first and favourite programs. As the name implies, it's a program that is used to keep schedules. You can have multiple schedules, with different frequencies, such as Daily, Weekly, and One-time. Whenever the time for a scheduled event approaches, you will receive a notification on your system.

Note: I have an updated version that is functional, but I want to add in an update feature. Until I find time to add in the feature, I'm keeping the updated one hidden.

Repo: Schedule Keeper
